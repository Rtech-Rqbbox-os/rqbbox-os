Embed RQBBOX OS Info Card

Add the RQBBOX OS information card to any website, blog, or landing page.
Method 1 — One-line Script Embed (Recommended)

Works on any HTML site including WordPress (Custom HTML block), Shopify, Squarespace, Wix, Weebly, and static sites. Just paste this one line anywhere in your page:

<script src="https://rtech-rqbbox-os.github.io/rqbbox-os/System/Website/embed-card.js"></script>
Method 2 — iframe Embed (All Platforms)

Works everywhere — Medium, Notion, WordPress.org self-hosted, Ghost, Jekyll, Hugo, and any site that supports iframes:

<iframe src="https://rtech-rqbbox-os.github.io/rqbbox-os/System/Website/embed-card.html" width="100%" height="480" frameborder="0" style="border-radius:16px;max-width:600px;margin:0 auto;display:block;" allowfullscreen></iframe>
Method 3 — Browser Extensions (FREE — Google Search Injection)

Install the RQBBOX OS browser extension and the info card appears directly in Google Search results when you search for RQBBOX OS. Works on google.com and googlechrome.com — no need to wait for Google to index the knowledge panel. All options are FREE — no developer fee required.

🦊 Firefox (FREE store, no fee):
1. Go to addons.mozilla.org → submit the Firefox extension ZIP
2. No registration fee — completely free to publish
3. Anyone can install from Firefox Add-ons

🧩 Tampermonkey Userscript (ALL browsers):
1. Install Tampermonkey extension (free, any browser)
2. Click the .user.js link → Install
3. Works on Chrome, Firefox, Edge, Opera, Brave

🌐 Chrome (Developer Mode):
1. Download the ZIP below
2. Go to chrome://extensions/ → Developer mode → Load unpacked
3. Search "RQBBOX OS" on google.com — card appears instantly
