About

RQBBOX OS is a portable USB-based gaming operating system developed by RhysTech. It runs entirely from a USB drive without requiring installation, emulation, or rebooting the host machine. The OS uses a Node.js HTTP server and full-screen browser-based launcher with a PS5-inspired interface, pro synthesized audio engine, and support for 43+ store packages including native games, web apps, and full Google Play Store integration.

RQBBOX OS is designed for gamers who want a portable, no-installation gaming environment they can carry on a USB drive and use on any compatible computer — Windows, macOS, Linux, Android, iOS, or KaiOS.
