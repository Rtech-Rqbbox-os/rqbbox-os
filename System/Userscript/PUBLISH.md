HOW TO PUBLISH TAMPERMONKEY USERSCRIPT (FREE - NO FEE)
======================================================
Developer: rhyscotton20@gmail.com

The Tampermonkey script is the EASIEST way for anyone to get the RQBBOX OS
info card on Google Search. Works on Chrome, Firefox, Edge, Opera, Brave.

How users install (no developer account needed for YOU):
--------------------------------------------------------
1. Install Tampermonkey (free) from their browser's extension store:
   - Chrome: https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo
   - Firefox: https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/
   - Edge: https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd

2. Click the userscript link:
   https://github.com/Rtech-Rqbbox-os/rqbbox-os/raw/main/System/Userscript/rqbbox-os-google-card.user.js

3. Tampermonkey opens and asks to install — click "Install"

4. Search "RQBBOX OS" on Google — card appears instantly

For publishing on Greasy Fork (optional, more visibility):
----------------------------------------------------------
1. Go to https://greasyfork.org/en/users/sign-in
2. Sign in with your GitHub account (free)
3. Go to https://greasyfork.org/en/scripts/new
4. Upload or paste the .user.js file
5. Fill in description, set license to MIT
6. Click "Post Script"
7. Link: https://greasyfork.org/scripts/[script-id]
