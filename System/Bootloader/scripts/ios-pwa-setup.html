📱 RQBBOX — iOS / iPadOS PWA Setup

No App Store needed. RQBBOX runs as a Progressive Web App on any iPhone or iPad running iOS 14.5+.

Prerequisites

1. RQBBOX server must be running on another device (PC, Android via Termux, or a Raspberry Pi)

2. Both devices must be on the same Wi-Fi network

3. iOS 14.5 or later recommended for full PWA features

Step-by-Step Setup

1 Find your server IP

On the device running RQBBOX server, note the IP address shown when the server starts (e.g., http://192.168.1.100:19777)

2 Open Safari

Open Safari on your iPhone/iPad (Chrome/Firefox cannot install PWAs on iOS)

3 Navigate to RQBBOX

Type your server's IP address in the address bar:
http://192.168.1.100:19777

The RQBBOX Phone Bootloader page will load.

4 Add to Home Screen

Tap the Share button (square with arrow at the bottom of the screen)

Scroll down and tap Add to Home Screen

Customize the name (default: "RQBBOX Phone Bootloader")

Tap Add in the top-right corner

5 Launch RQBBOX

RQBBOX icon appears on your home screen — tap to launch

It opens in full-screen mode (no Safari address bar)

Pro tip: Use the share sheet to send game links to friends

Additional Setup

+ Enable Notifications (iOS 16.4+)

Settings → Safari → Advanced → Web Notifications → Toggle ON

RQBBOX can send game invites and updates

+ Badge Numbers

Settings → Notifications → RQBBOX → Badges → ON

+ Background Audio

RQBBOX audio continues when phone is locked — enable in Settings → RQBBOX → Background Audio

Troubleshooting

❌ "Cannot connect to server"

• Ensure both devices are on the same Wi-Fi network

• Check the server is running (node System/Server/server.js)

• Try the IP 127.0.0.1:19777 if running server on the same device via iSH (iOS terminal)

❌ "PWA not installing"

• You MUST use Safari — Chrome/Firefox don't support iOS PWA installs

• iOS 14.5+ required

• Server must serve HTTPS or localhost — for HTTP on local network, ensure proper network config

❌ "App crashes on launch"

• Clear Safari cache: Settings → Safari → Clear History and Website Data

• Remove and re-add to Home Screen

• Update RQBBOX to the latest version
