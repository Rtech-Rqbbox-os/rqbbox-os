# Xiaomi — RQBBOX Bootloader Guide

## Supported Devices
- Xiaomi Mi series (Mi 1 through Mi 15)
- Redmi series (Redmi 1 through Redmi Note 14)
- POCO series (Poco F1 through Poco F7, X, M series)
- Xiaomi Pad/Pad 5/6/7
- Black Shark 1–5 (gaming phones)

## Boot Methods

### Method 1: Termux (Best)
1. Install Termux from F-Droid
2. Enable "Float Window" for Termux in **Second Space** or **Game Turbo**
3. `pkg install nodejs` → `node System/Server/server.js`

### Method 2: Game Turbo
1. Open **Security** app → **Game Turbo**
2. Add RQBBOX as a game
3. Configure:
   - CPU boost: On
   - GPU optimized: On
   - Touch response: Enhanced
   - Network: Dedicated
   - Voice changer: Optional
4. Launch from Game Turbo for max performance

### Method 3: MIUI Second Space
1. Settings → Second Space
2. Set up a gaming space
3. Move RQBBOX to Second Space
4. Switch between work and gaming profiles

### Method 4: Xiaomi Pad (Tablet)
1. Open in split-screen:
   - Sidebar gesture → drag RQBBOX to side
   - Use remaining space for game guides/discord

## Performance Tips
- Black Shark: "Shark Space" mode unlocks full 120Hz gaming + liquid cooling fans
- POCO: Use "POCO Launcher" with RQBBOX as main game hub
- Enable "Memory Extension" for better multi-tasking
- Disable MIUI optimization for Termux (Developer options)
- Redmi: Lite profile for RQBBOX — reduces render resolution for smoother gameplay
